# Google Sheets Operator Surface

Google Sheets is a collaboration/export surface, not source of truth.

Supported use cases:

- export opportunities,
- export search terms,
- export negative keyword candidates,
- export content plan,
- export social calendar,
- export product feed issues,
- import reviewed bulk edits,
- append client summary rows.

Sheets exports must include evidence IDs and action IDs. Imports must validate action IDs before applying anything.
