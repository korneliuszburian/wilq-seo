# Demand Gen Migration

Demand Gen is a first-class Ads Doctor module.

Diagnostics:

- display campaign inventory,
- migration candidates,
- asset readiness,
- image/video/carousel requirements,
- CTA variants,
- audience targeting,
- Google AI enhancements state,
- budget and goal alignment.

Rules:

- Detect Display campaigns needing migration planning.
- Detect missing or weak creative assets.
- Never silently enable Google AI enhancements.
- Expose readiness in API and dashboard route `/ads-doctor/demand-gen`.
